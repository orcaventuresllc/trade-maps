Testing if SVG Paths Render

If you see a blue shape below with red border around the SVG area, the paths are valid:

Debug info:

  • SVG dimensions: 800x500 (explicitly set)
  • ViewBox: 0 0 959 593
  • Red border shows SVG boundaries
  • Gray background shows SVG area
  • Blue shapes are the state paths
  • Hover over shapes to see them turn red

If you see NO blue shapes but DO see the gray rectangle with red border, the path data may be invalid or outside the viewBox.
